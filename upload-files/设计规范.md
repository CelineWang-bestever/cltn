# 美容师资料上传与管理系统 - 设计规范

## 1. 设计理念

采用轻奢简约风格，以豆沙粉为主色调，传达专业、温暖、值得信赖的品牌形象。

## 2. 设计语言

### 2.1 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #E8A0B4 | 豆沙粉，柔和温暖 |
| 主色深 | #D4879B | 深豆沙粉 |
| 主色浅 | #F5D5DE | 浅粉背景 |
| 辅助色 | #6B5B6E | 深紫灰 |
| 成功色 | #52C41A | 成功状态 |
| 警告色 | #FAAD14 | 警告状态 |
| 错误色 | #FF4D4F | 错误状态 |
| 文字主色 | #333333 | 主要文字 |
| 文字次色 | #666666 | 次要文字 |
| 文字弱色 | #999999 | 弱化文字 |
| 背景色 | #FAFAFA | 页面背景 |
| 边框色 | #EEEEEE | 边框线条 |

### 2.2 字体系统

- 主字体：PingFang SC, -apple-system, BlinkMacSystemFont, sans-serif
- 标题字号：18px-24px
- 正文字号：14px-16px
- 辅助字号：12px

### 2.3 间距系统

- 基础单位：4px
- 常用间距：8px, 12px, 16px, 20px, 24px, 32px
- 卡片内边距：16px
- 页面边距：16px (移动端), 24px (Pad端), 32px (PC端)

### 2.4 圆角系统

- 小圆角：4px
- 中圆角：8px
- 大圆角：12px
- 全圆角：50%

## 3. 组件规范

### 3.1 按钮

**主要按钮**
- 背景：#E8A0B4
- 文字：#FFFFFF
- 高度：44px (移动端), 48px (Pad端), 40px (PC端)
- 圆角：8px
- Hover：#D4879B

**次要按钮**
- 背景：#FFFFFF
- 文字：#E8A0B4
- 边框：1px solid #E8A0B4
- Hover：背景#FFF5F7

**危险按钮**
- 背景：#FF4D4F
- 文字：#FFFFFF

### 3.2 上传组件

**上传区域**
- 边框：2px dashed #DDDDDD
- 背景：#FAFAFA
- 圆角：8px
- Hover：边框#E8A0B4，背景#FFF5F7

**上传预览卡片**
- 宽度：(100% - 间距) / n
- 高度：100px
- 圆角：8px
- 封面图填充方式：cover

### 3.3 列表项

**资料列表项**
- 背景：#FFFFFF
- 高度：72px
- 边框：底部1px #EEEEEE
- Hover：背景#FFF8FA

**操作按钮**
- 尺寸：32px x 32px
- 图标大小：18px

### 3.4 Tab切换

**Tab标签**
- 高度：44px
- 选中态：文字#E8A0B4，底部2px #E8A0B4
- 未选中：文字#666666

### 3.5 筛选器

**下拉选择器**
- 高度：36px
- 边框：1px solid #EEEEEE
- 圆角：6px

**日期选择器**
- 样式：平铺式日期选择
- 选中态：背景#E8A0B4

## 5. 功能流程

### 5.1 客户资料上传流程

```
1. 选择客户 → 2. 选择资料类型(客户资料) → 3. 拍照/选择相册
      ↓
4. 文件校验(仅图片) → 5. 填写备注 → 6. 确认上传
      ↓
7. 保存记录 → 8. 显示成功提示
```

### 5.2 每日经营数据上传流程

```
1. 选择客户 → 2. 选择资料类型(每日经营) → 3. 拍照/选择相册/选择文件
      ↓
4. 文件校验(图片+文件) → 5. 填写备注 → 6. 确认上传
      ↓
7. 保存记录 → 8. 显示成功提示
```

### 5.3 历史记录查看流程

```
1. 进入历史记录 → 2. 选择筛选条件(日期/操作人/类型)
      ↓
3. 点击搜索 → 4. 查看分页列表 → 5. 点击查看预览
      ↓
6. 关闭预览 → 返回列表
```

## 6. 状态定义

### 6.1 上传状态

| 状态 | 显示 |
|------|------|
| 待上传 | 灰色虚线框 + 上传图标 |
| 上传中 | 蓝色进度环 |
| 上传成功 | 绿色勾 + 缩略图 |
| 上传失败 | 红色感叹号 + 重试按钮 |

### 6.2 空状态

- 无历史记录：显示空状态插图 + "暂无上传记录"
- 筛选无结果：显示"未找到符合条件的记录"

### 6.3 加载状态

- 列表加载：骨架屏占位
- 图片加载：灰色占位 + 加载动画

## 7. 权限矩阵

| 功能 | 店总 | 店长 | 美容师 |
|------|------|------|--------|
| 查看所有历史记录 | ✓ | ✓ | ✗ |
| 查看自己上传的每日经营数据 | ✓ | ✓ | ✓ (Pad端) |
| 查看所有客人资料 | ✓ | ✓ | ✓ (Pad端) |
| 上传资料 | ✓ | ✓ | ✓ |
| 删除资料 | ✓ | ✓ | 仅自己上传 |

## 8. 技术约束

### 8.1 文件限制

| 类型 | 格式 | 大小 | 数量 |
|------|------|------|------|
| 图片 | jpeg, png, jpg | ≤10MB/张 | ≤10个/次 |
| 文件 | pdf, word | ≤20MB/个 | ≤10个/次 |

### 8.2 分页规则

- 每页显示：20条
- 页码导航：最多显示7个页码按钮

### 8.3 预览支持

- 图片：支持放大、缩小
- PDF：显示为封面 + 页数
- Word：转换为PDF后预览
