## 详细弹窗规格

### 4.1 会员选择弹窗

| 属性 | 说明 |
|-----|------|
| 弹窗ID | `memberModalOverlay` |
| 实现方式 | iframe 加载 `popup-find-customer.html` |
| 关闭函数 | `hideMemberModal()` |
| 核心程度 | **核心** |

**交互流程**:
1. 点击"选择会员"按钮
2. 弹窗以iframe加载会员列表页面
3. 选择会员后关闭弹窗，更新页面会员信息

---

### 4.2 客户权益弹窗

| 属性 | 说明 |
|-----|------|
| 弹窗ID | `benefitModalOverlay` |
| 打开函数 | `openBenefitModal()` |
| 关闭函数 | `hideBenefitModal()` |
| 核心程度 | **核心** |

**功能列表**:
- 搜索会员已有权益/卡项
- 最近搜索记录（最多8条）
- 查看卡项详情（余额、赠金、折扣规则）
- 使用权益
- 升卡操作
- 充卡操作

**权益卡数据结构**:
```javascript
{
    id: 'card-001',
    type: '有限次卡',  // 有限次卡 / 金额卡
    name: '面部护理卡',
    validDate: '永久有效',
    benefits: [
        { id: 'b-001', name: '深度放松精油SPA', used: 0, total: 3, isGift: false }
    ]
}
```

**按键布局**:
```
┌─────────────────────────────┐
│  设置数值                [×] │
├─────────────────────────────┤
│           0.00              │
├─────────────────────────────┤
│  [1]  [2]  [3]  [←]        │
│  [4]  [5]  [6]  [×]        │

└─────────────────────────────┘
```

**按键功能**:
- 数字键 (0-9): 输入数字
- `.`: 小数点
- `←`: 退格
- `×`: 关闭
- `✓`: 确认

## 弹窗通用交互规范

### 显示/隐藏控制
```javascript
// 显示弹窗
overlay.classList.add('show');

// 隐藏弹窗
overlay.classList.remove('show');
```

### Visual Viewport 适配
部分弹窗支持 Visual Viewport API 进行位置调整，防止键盘弹出时遮挡内容。

### 遮罩层点击关闭
弹窗支持点击遮罩层关闭（部分关键弹窗除外）。
