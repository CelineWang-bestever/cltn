# 与竞品能力差异概述

本文梳理与镜美人等竞品相比存在差异的能力点，按业务页面归类。

| 业务页面     | 功能点                                                     | 参考竞品    | 是否需要后端配合升级 | 与回蓝差异                                                   |
| -------- | ------------------------------------------------------- | ------- | ---------- | ------------------------------------------------------- |
| **会员详情** | 会员详情页露出明显的开单入口                                          | 镜美人等竞品  | 否          | -                                                       |
| **会员详情** | 会员详情页客户权益平铺展示，并支持在当前页对已有权益进行快速消耗                        | 镜美人     | 否          | -                                                       |
| **会员详情** | 展示消费画像与账户价值概览                                           | 竞品均支持   | 是          | -                                                       |
| **会员详情** | 账户价值概览、钱包明细页面充值、退款、欠款明细页面、还款功能                          | -       | -          | 回蓝无。已有接口能力：钱包明细页面、钱包充值&退款功能、欠款还款功能。Pad端\&PC端需要新增欠款流水页面。 |
| **开单页**  | 查找会员交互优化                                                | 博卡      | 否          | -                                                       |
| **开单页**  | 开单时展示关联客户欠款                                             | 竞品均支持   | 是          | 回蓝无                                                     |
| **开单页**  | 聚合多种商品类型并统一开单                                           | -       | 是          | -                                                       |
| **开单页**  | 下单交互对标镜美人：通过商品弹窗选购商品；已选商品支持录入"产品拿走数量"；剩余未拿走部分可直接寄存至客人名下 | 镜美人     | 否          | -                                                       |
| **开单页**  | 购买单次护理支持选择本次消耗数量；未消耗的单次护理在客户详情页展示                       | 镜美人     | 是          | -                                                       |
| **开单页**  | 已选商品行支持复制，减少重复添加选购商品的操作                                 | 镜美人     | 否          | -                                                       |
| **开单页**  | 单件商品支持多张金额卡划扣                                           | 有赞      | 是          | 回蓝无                                                     |
| **开单页**  | 金额卡充值支持录入赠金                                             | 竞品均支持   | 是          | 回蓝无                                                     |
| **开单页**  | 金额卡充值支持赠送权益                                             | 有赞      | 是          | 回蓝无                                                     |
| **开单页**  | 升级为金额卡，快速定义金额卡售卡价格、本金、赠金                                | -       | -          | 回蓝无                                                     |
| **开单页**  | 配料明细支持收起展示                                              | 博卡      | 否          | -                                                       |
| **开单页**  | 整单改价交互对标有赞                                              | 有赞      | 否          | -                                                       |
| **开单页**  | 定制会员权益                                                  | 镜美人     | 是          | 回蓝无                                                     |
| **开单页**  | 添加居家产品时商品列表展示升级                                         | 镜美人     | 是          | -                                                       |
| **开单页**  | 支持机器人与非机器人项目合并下单                                        | 无可参考的竞品 | -          | 回蓝无                                                     |
| **开单页**  | 每件商品支持增加备注                                              | 镜美人     | 是          | -                                                       |
| **支付页**  | 支持组合支付                                                  | 竞品均支持   | 是          | 回蓝无                                                     |
| **支付页**  | 展示金额卡划扣金额                                               | 有赞      | 是          | 回蓝无                                                     |

<br />

<br />

**说明：**

- 表格按业务页面分类，清晰展示各功能点的参考竞品、后端配合需求及与回蓝的差异
- 标为"回蓝无"的功能表示该功能在回蓝系统中不存在
- "是否需要后端配合"列标注为"是"的功能需要后端进行相应升级支持
