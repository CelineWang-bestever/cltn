# 03 - 已选商品管理

> **所属模块**: 开单收银页
> **功能点数**: 13项
> **核心程度**: ⭐⭐⭐ 核心

---

## 功能清单

### 3.1 单次护理区域

| 属性 | 说明 |
|-----|------|
| 功能描述 | 已选单次疗程表格 |
| 表格ID | `service-table` / `service-table-body` |
| 核心程度 | **核心** |

**表格列定义**:

| 列名 | CSS类 | 说明 |
|-----|-------|------|
| 赠 | `col-gift-flag` | 赠品标记复选框 |
| 商品名称 | `col-name` | 商品名称 |
| 原价 | `col-original` | 商品原价 |
| 购买 | `col-qty` | 购买数量步进器 |
| 本次消耗 | `col-consume` | 本次消耗数量 |
| 使用卡项 | `col-card` | 使用哪个卡项（仅会员可见） |
| 折扣 | `col-discount` | 折扣百分比输入 |
| 折后价总金额 | `col-final` | 折后价计算结果 |
| 美容师 | `col-beautician` | 分配美容师 |

### 3.9 数量步进器

| 属性 | 说明 |
|-----|------|
| 功能描述 | 购买数量/消耗数量的 +/- 控件 |
| 实现方式 | 自定义步进器组件 `table-stepper` |
| 核心程度 | UI交互 |

**实现结构**:
```html
<div class="table-stepper">
    <div class="table-stepper-btn" onclick="tableStepDown(this)">-</div>
    <input type="number" class="table-stepper-value" value="1" min="0" max="99">
    <div class="table-stepper-btn" onclick="tableStepUp(this)">+</div>
</div>
```

## 分类区域控制

| 区域 | 显示条件 | 控制元素 |
|-----|---------|---------|
| 单次护理 | 有已选单次护理时显示 | `#serviceCategorySection` |
| 产品 | 有已选产品时显示 | `#productCategorySection` |
| 金额卡 | 有已选金额卡时显示 | `#amountCardCategorySection` |
| 疗程 | 有已选疗程卡时显示 | `#courseCategorySection` |

**空状态提示**:
- 主空状态: "请从左侧【添加商品】区域点击菜单开始选购"
- 各类别空状态: "暂无已选xxx"

## 表格数据结构

每行数据通过 `data-*` 属性存储：

```javascript
// 单次护理/产品行
tr.setAttribute('data-item-id', itemId);
tr.setAttribute('data-item-type', 'service' | 'product' | 'course' | 'amountCard');

// 权益消耗行
tr.setAttribute('data-benefit-id', benefitId);
tr.setAttribute('data-card-id', cardId);
```
