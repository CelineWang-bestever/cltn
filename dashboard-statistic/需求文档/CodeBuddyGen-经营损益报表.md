# 经营损益报表 — 需求文档

## 1. 需求概述

| 项目       | 内容                                                                                                             |
| -------- | -------------------------------------------------------------------------------------------------------------- |
| **需求类型** | 新增功能 — 门店经营损益分析与可视化                                                                                            |
| **业务背景** | 克丽缇娜门店需要定期核算经营损益，通过汇总护理服务消耗收入、产品销售收入与各项经营支出，计算门店月度/年度利润，辅助店长和总部进行经营决策。当前损益表需整合"消耗业绩（收入）"和"支出费用"两大模块，形成完整的利润视图。 |
| **核心目标** | 1. 按月展示门店经营损益全貌（收入、支出、利润）；2. 提供多级下钻明细，从汇总到品类再到具体项目；3. 可视化展示支出结构分布和利润趋势变化；4. 支持按月份切换和历史数据对比；5. 提供数据导出能力。        |

## 3. 术语定义

| 术语        | 英文                             | 说明                                        |
| --------- | ------------------------------ | ----------------------------------------- |
| 经营损益表     | Profit & Loss Statement (P\&L) | 反映门店在某期间内的收入、支出及利润的财务报表                   |
| 消耗业绩      | 消耗业绩 = 护理消耗 + 产品消耗 − 本月退款的金额   | <br />                                    |
| 护理消耗      | Treatment Consumption          | 护理服务消耗产生的收入（具体收入口径见 4.3）                  |
| 产品消耗      | Product Consumption            | 居家产品消耗产生的收入（具体收入口径见 4.3）                  |
| 总支出费用     | Total Expense                  | 门店当月全部经营支出总和                              |
| 本月退款的订单商品 | Monthly Refunded Items         | 当月发生退款的单次护理/居家产品退款金额（按负收入口径处理，详见 4.3/4.4） |
| 利润        | Profit                         | 消耗业绩 − 总支出费用                              |
| 利润率       | Profit Margin                  | (利润 ÷ 消耗业绩) × 100%                        |

## 4. 业务收入详解

### 4.1 消耗业绩构成

```
一、消耗业绩（收入）
├── 一.1 护理消耗
│   ├── 耗卡订单中的护理服务耗卡金额
│   └── 单次护理消耗（收入口径见 4.3；消耗数量取 m_master_order_item_detail.goods_use_qty）
│   └── 按商品分类消耗金额降序排列
│
├── 一.2 产品消耗
│   ├── 耗卡订单中的居家产品耗卡金额
│   └── 居家产品消耗（收入口径见 4.3；消耗数量取 m_master_order_item_detail.goods_use_qty）
│
└── 退款扣减：退款明细以负数并入对应品类（护理消耗/产品消耗）下的最末级行，不单独展示“减：本月退款”汇总行
```

### 4.2 关键业务规则（索引）

- 收入口径（有效/撤销、开卡、赠送、退款、退卡、渠道）：见 4.3
- 本年利润Hero口径、明细表占比基准、退款展示位置、支出下钻、导出、趋势图与空态：见 4.4

### 4.3 收入口径补充（正向/逆向订单）

本节为最终口径说明，如与上文规则存在冲突，以本节为准。

**正向订单**

- 仅取状态有效的订单，撤销状态的订单不取
- 开卡（开金额卡、开次卡）：不记为收入
- 单次护理消耗
  - 从卡项中消耗单次护理：以消耗卡项的金额记为收入；赠品因消耗金额为 0，不记为收入
  - 客户直接购买单次护理：记为收入；若设置为赠送，支付金额为 0，不记为收入；客户使用金额卡划扣，也记为收入
- 居家产品消耗
  - 从卡项中消耗居家产品：以消耗卡项的金额记为收入
    - 补充说明：目前卡项中居家产品只能作为赠送权益，目前无收入；后续开单收银优化后，次卡的购买权益允许添加居家产品，遵循当前规则，消耗卡项中的居家产品记为收入
  - 客户直接购买居家产品：记为收入；若设置为赠送，支付金额为 0，不记为收入；客户使用金额卡划扣，也记为收入
- 单次护理和居家产品消耗的收入：不看订单来源，核销、云店等其他非店务通渠道下单的订单，都记为门店收入

**逆向订单**

- 退卡：不需要从收入部分扣除
- 单次护理退款金额：从收入部分扣除（以负收入统计）
- 居家产品退款金额：从收入部分扣除（以负收入统计）

### 4.4 计算与展示口径补充（统一口径）

本节用于沉淀“计算口径 + UI展示”规则，避免在多处重复描述。

- 本年利润Hero：本年为当前选中月份所在自然年，按 1 月至该月累计；切换月份时同步刷新对应年份的累计数据
- 明细表占比列：统一以“总消耗业绩”为基准（可出现 >100% 或负值）
  - 示例：总消耗业绩 100，支出 120，利润 -20 → 支出占比 120%，利润占比 -20%
- 退款扣减展示：退款明细以负数并入对应品类（护理消耗/产品消耗）下的最末级行，不单独展示“减：本月退款”汇总行
- 支出费用下钻：不使用弹窗；支出类别展开后直接展示多条支出明细记录
- 日期 + 经办人：仅支出费用的最末级明细记录展示；消耗业绩为聚合后的商品明细，不展示日期/经办人
- 趋势图（近6个月）：以当前选中月份为终点，展示包含该月份在内的最近 6 个月；若跨年度自动补齐上一年数据；当前选中月份使用特殊颜色高亮；Hover 提示展示“当月利润金额 + 环比（相对上月）百分比”
- 导出规格
  - 导出范围：当前选中月份
  - 文件格式：XLS
  - 文件名规则：`损益报表_YYYY年M月.xls`（示例：`损益报表_2026年5月.xls`）
  - 内容结构：概要、消耗明细、支出明细（含备注）
  - 排序规则：消耗明细按商品分类消耗金额降序；支出明细按时间倒序
  - 金额展示：所有金额统一保留 2 位小数，单位为元
- 空态与缺数
  - 明细表无数据：显示“暂无数据”
  - 某月无数据：趋势图按 0 展示该月柱子
  - 消耗业绩为 0：利润率显示 0.0%；支出结构分析（环形图）显示空态占位
- 支出结构环形图“其他”规则：单项占比 < 1% 的类别合并为“其他（X项）”

## 5. 业务流程分析

### 5.1 主流程 — 查看损益报表

```
[店长/财务] → 进入经营损益表页面
   → 默认展示当前月份数据
   → 查看本月利润Hero区（利润金额 + 消耗业绩 + 支出费用）
   → 查看本年利润Hero区
   → 查看损益明细表：
       - 消耗业绩（展开查看护理消耗/产品消耗 → 子项目；退款明细按负数并入对应品类）
       - 总支出费用（展开查看各类别 → 支出明细记录）
       - 利润行
   → 查看图表：
       - 支出结构分析（环形图）
       - 月度利润变化（柱状趋势图）
   → 可选：切换月份，数据联动刷新
   → 可选：导出当前月份报表
```

### 5.2 子流程 — 损益明细表展开/收起

```
[损益明细表]
   → 一级行（level-0）："一、消耗业绩（收入）" / "二、总支出费用" + 展开按钮
   → 点击展开按钮 → 显示二级行（level-1）：如"护理消耗"、"产品消耗"、"日常运营成本"等
   → 二级行可再次展开 → 显示最末级行（level-2）：
       - 消耗业绩：展示该类别下的消耗明细记录（含退款明细负数行，按明细并入对应品类）
       - 支出费用：展示该类别下的所有支出明细记录（按时间倒序），若同一项目有多条记录则逐条展示，不合并
   → level-2 行显示：
       - 消耗业绩（聚合后）：商品分类名称、金额、占比（不展示日期/经办人）
       - 支出费用（明细记录）：项目名、金额、占比、日期、经办人
   → 点击"展开全部/收起全部" → 全部展开或全部收起
```

### 5.3 子流程 — 查看消耗业绩明细（表内展开）

```
[损益明细表]
   → 点击展开二级行"护理消耗"或"产品消耗"
   → 直接在表格内展示该类别下的消耗明细记录（不使用弹窗）
   → 每条明细记录为最末级行（level-2）：显示商品分类名称、消耗金额、占比（不展示日期/经办人）
   → 若同一项目有多条记录，则逐条展示，不合并
```

### 5.4 子流程 — 查看支出费用明细（表内展开）

```
[损益明细表]
   → 点击展开二级行"日常运营成本"等支出类别行
   → 直接在表格内展示该类别下多条支出明细记录（不使用弹窗）
```

### 5.5 子流程 — 按月份筛选

```
[页面顶部]
   → 月份下拉选择器（2026年1月 ~ 6月）
   → 切换月份后联动刷新：
       - 本月利润Hero区数据
       - 本年利润Hero区数据（按 1 月至该月累计）
       - 损益明细表数据
       - 支出结构分析图
       - 月度利润趋势图
```

### 5.6 子流程 — 导出报表

```
[页面顶部]
   → 点击"导出报表"
   → 按当前选中月份过滤数据
   → 导出规格见 4.4
   → 触发浏览器下载
```

## 7. 关键场景覆盖

### 7.1 正向场景

- 店长查看5月份损益表：本月利润 ¥22,480.00（利润率 23.9%），消耗业绩 ¥94,180.00（含退款扣减 -¥1,200.00），支出 ¥71,700.00
- 展开日常运营成本细项，看到房租 ¥15,000.00（占比15.7%）等明细
- 查看支出结构环形图：人员支出占比最大（49.4%），其次是日常运营（35.9%）
- 看到月度利润趋势稳步上升，从12月的 ¥18,200.00 增长到5月的 ¥22,480.00

### 7.2 对称场景（逆向操作）

- 展开 ↔ 收起（单行/全部）
- 切换月份 ↔ 数据刷新

### 7.3 边界条件

- 月份范围：默认选中当月，支持修改月份，最大月份为当前月份，无最小月份限制
- 利润计算：消耗业绩 − 支出费用，允许为负数（亏损）
- 利润率：消耗业绩为0时不计算（除零保护）
- 支出明细中单条金额：最小0.01元（1分），所有金额统一保留2位小数
- 环形图占比之和 = 100%

### 7.4 异常处理

| 异常场景        | 处理方案                                         |
| ----------- | -------------------------------------------- |
| 选中月份无支出数据   | 支出为0，利润 = 消耗业绩                               |
| 选中月份无消耗业绩   | 消耗业绩为0，利润为负值（= −支出）                          |
| 本月有大额退款     | 退款明细以负数并入对应品类，不影响支出侧（详见 4.4）                 |
| 利润为负        | 金额显示为负数并用红色；支出结构环形图仅基于支出费用计算，不受利润正负影响，始终正常展示 |
| 消耗业绩为0计算利润率 | 固定显示"0.0%"避免除零错误                             |
| 导出时数据量过大    | 按月份过滤导出，避免超过浏览器内存限制                          |

### 7.5 多样性场景

- **月份多样性**：支持跨年度数据（当前演示为2026年1\~6月）
- **品类多样性**：护理消耗下多个疗程类别（补水、抗衰、清洁等），产品消耗下多个产品类别（面膜、精油等）
- **支出类别多样性**：6大支出类别，每个类别下多个具体项目
