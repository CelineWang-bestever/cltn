# dashboard-statistic 指标卡与表格清单

## 1-dashboard-overview\.html

### 指标卡（含口径/tooltip）

**经营分析**

- 现金业绩：订单完成时间在统计时间内，客户实际付款总金额，统计时减去退款金额
- 产品现金：统计时间内，产品类产生的现金收入
- 护理现金：统计时间内，护理类产生的现金收入
- 卡项现金：统计时间内，卡项销售产生的现金收入（含开卡/充值/升级等）
- 新客现金业绩：统计时间内，新客产生的现金收入总额
- 新客产品现金：统计时间内，新客产生的产品现金收入
- 新客护理现金：统计时间内，新客产生的护理现金收入
- 新客卡项现金：统计时间内，新客产生的卡项现金收入（含开卡/充值/升级等）
- 消耗业绩：\
  1.订单完成时间在统计时间内，客户消耗总金额，统计时减去退款金额及客户消耗回退金额；

  2.不含赠送权益的消耗；
- 护理消耗：统计时间内，卡项中护理类权益的消耗金额合计
- 产品消耗：统计时间内，卡项中产品类权益的消耗金额合计
- 客单价：统计时间内，总现金业绩平均每单成交金额

**客户分析**

- 新增新客人数：统计时间内新增的新客数量（没有消费过）
- 新增体验客人数：统计时间内新增的体验客数量（产生消费，但交易记录中不包含卡项）
- 新增会员人数：统计时间内新增的会员数量（产生消费，且交易记录中包含卡项）
- 客流量：订单完成时间在统计时间内，付款的人次（同一客户按日去重跨天累加，散客按散客订单数统计）
- 到店客户数：订单完成时间在统计时间内，付款的客户数（同一客户统计时间内去重，散客按散客订单数统计）
- 服务项次数：统计时间内，提供给客户的服务项次数总计（可展开订单明细）
- 欠款（当前）：当前存量指标，不随时间筛选变化（客户欠款合计）
- 金额卡资产总额（当前）：当前存量指标，不随时间筛选变化（金额卡余额资产总额）
- 次卡资产总额（当前）：当前存量指标，不随时间筛选变化（次卡剩余价值资产总额）

**平台核销**

- 美团/大众点评核销：统计时间内，美团/大众点评渠道核销单数
- 抖音核销：统计时间内，抖音渠道核销单数
- 核销金额：统计时间内，平台核销金额合计

**预约到店**

- 预约人数：统计时间内产生的预约人数
- 实际到店：统计时间内实际到店的人数
- 到店率：实际到店 / 预约人数（不展开明细）

### 明细表表头（点击可展开的指标卡）

**预约到店**

- 预约人数：客户姓名 / 手机号 / 预约项目 / 责任美容师 / 预约时间 / 状态
- 实际到店：客户姓名 / 手机号 / 预约项目 / 责任美容师 / 预约时间 / 实际到店

**客户分析**

- 客流量：会员姓名 / 手机号 / 客户类型 / 最近到店 / 累计消费 / 责任美容师 / 到店时间
- 新增新客人数：客户姓名 / 手机号 / 客户类型 / 最近到店 / 累计消费 / 责任美容师 / 来源渠道 / 创建时间
- 新增体验客人数：客户姓名 / 手机号 / 客户类型 / 最近到店 / 累计消费 / 责任美容师 / 来源渠道 / 创建时间
- 新增会员人数：客户姓名 / 手机号 / 客户类型 / 最近到店 / 累计消费 / 责任美容师 / 来源渠道 / 创建时间
- 到店客户数：客户姓名 / 手机号 / 客户类型 / 交易次数 / 累计消费 / 责任美容师
- 服务项次数：客户姓名 / 手机号 / 客户类型 / 服务项目 / 数量 / 美容师 / 服务时间
- 金额卡资产总额：客户姓名 / 手机号 / 客户类型 / 金额卡余额 / 责任美容师 / 最近到店
- 次卡资产总额：客户姓名 / 手机号 / 客户类型 / 次卡剩余价值 / 次卡余次 / 赠送余次 / 责任美容师 / 最近到店
- 欠款：客户姓名 / 手机号 / 客户类型 / 当前欠款金额 / 已还款金额 / 历史欠款总金额 / 最近到店 / 责任美容师

## no-use-2-dashboard-spreadsheet.html

### 指标卡

- 无独立可展开指标卡；主内容为报表表格

### 报表表头

- 日维度表头（THEAD）：日期 + 现金业绩(合计/产品现金/护理现金/卡项现金) + 消耗业绩(合计/疗程消耗/产品消耗) + 客流(0护理客流/非0护理客流/产品客流/总客流) + 会员(体验人数/成交人数)
- 月维度表头（THEAD\_YEAR）：月份 + 现金业绩(合计/产品现金/护理现金/卡项现金) + 消耗业绩(合计/疗程消耗/产品消耗) + 客流(0护理客流/非0护理客流/产品客流/总客流) + 会员(体验人数/成交人数)

## 4-dashboard-staff-statistics.html

### 指标卡

- 现金业绩（含目标与完成率）
- 消耗业绩（含目标与完成率）
- 客流分析（服务人数/服务项数/客单价）
- 现金业绩明细（产品现金/护理现金/卡项现金/合计等分项展示，非表格）
- 消耗业绩明细（疗程消耗/产品消耗/合计等分项展示，非表格）

### 明细表表头（员工业绩表格）

- 分组表头（两行）：
  - 第一行：{日期/时段/月份} + 现金业绩 + 消耗业绩 + 服务人数 + 新增会员
  - 第二行（展开后的列）：
    - 现金业绩：合计 / 产品现金 / 护理现金 / 卡项现金
    - 消耗业绩：合计 / 疗程消耗 / 产品消耗
    - 服务人数：0护理客流 / 非0护理客流 / 产品客流 / 总客流
    - 新增会员：体验人数 / 成交人数

## 5-sales-goal-mgmt.html

### 指标卡

- 门店目标区：按月份的目标卡（每月目标/已完成/操作入口）
- 员工目标区：按目标类型渲染的目标卡（每张卡显示目标、完成等信息）

### 表格

- 本页主要为卡片与表单弹窗交互；未使用固定 `<table><thead>` 的静态表头结构（因此“表头”不适用）
